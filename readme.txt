Welcome to Showers of Bombs

In this game, the user must avoid bombs that drop every the sky. Every time the paddle gets hit by a bomb, the user loses all their points and their score is set to 0. The user also begins the game with ten lives. Every time the ball hits the bottom of the screen the user loses one life. 

The user gains points by using the paddle to hit the ball (each hit earns them 20 points). To win the game, the user must either gain a 100 points or survive the entire 60 seconds of the game
